Miten tyhjän tagin käyttö eroaa muiden tagien käytöstä?

Tyhjät tagit eivät sisällä sisältöä eivätkä tarvitse sulkevaa tagia, esim. <br> tai <img>.
Tavalliset tagit sisältävät sisältöä ja tarvitsevat sulkevan tagin, esim. <p>Teksti</p>.

Mitä Case sensitive tarkoittaa?

"Case sensitive" tarkoittaa, että isot ja pienet kirjaimet lasketaan eri merkeiksi. Esim. A ≠ a.

Miten var, let ja const eroavat toisistaan.

var → vanha, function scope, voi julistaa uudestaan.
let → uusi, block scope, arvoa voi muuttaa mutta ei julistaa uudestaan samassa lohkossa.
const → block scope, arvoa ei voi muuttaa (mutta objektin/taulukon sisältöä voi).
